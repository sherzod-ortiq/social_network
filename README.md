# Social network app
### The app was built in tutorial

### Tech stack info:

* Rails version: 5.1.6.2
* Default development database: Sqlite3
* Automatic test framework: Minitest

### Steps for running the app:

	1. bundle install
	2. rails db:create
	3. rails db:migrate
	4. rails server

### Additional settings
	1. rails test # run auto tests with Minitest
